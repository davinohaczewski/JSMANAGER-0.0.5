JSManager

Seja bem vindo(a) ao site informativo do JSManager.

Oferecemos

O JSManager é um modelo de servidor para pequenas empresas e escritórios seguirem, realizando a implementação de uma solução de baixo custo para fazer um gerenciamento de sua rede local.

O nosso modelo utiliza como base o Ubuntu Server, um software de uso livre, onde podemos modificar o código fonte permitindo criar um ambiente de gestão.
